JavaScript 单元测试
===

- jasmine
- karma
- istanbul

## Install

```sh
$ npm install -g jasmine
$ npm install -g karma
$ npm install -g karma-coverage
```

## Config

```sh
$ karma init
```

- karma.conf.js

## Run

- jasmine
- karma `karma start karma.conf.js`
- istanbul `istanbul check-coverage --statement -5 --branch -3 --function 100`
